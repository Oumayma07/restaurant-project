# 🍽 Ô Délices – Application de Gestion de Restaurant

[![Angular](https://img.shields.io/badge/Angular-16-red)](https://angular.io/)
[![Spring Boot](https://img.shields.io/badge/Spring%20Boot-3-brightgreen)](https://spring.io/projects/spring-boot)
[![MongoDB](https://img.shields.io/badge/MongoDB-Database-blue)](https://www.mongodb.com/)
[![License](https://img.shields.io/badge/License-MIT-blue)](LICENSE)

Une application **full-stack** pour la gestion d’un restaurant, développée avec **Angular** pour le frontend et **Spring Boot / MongoDB** pour le backend.  
Elle permet aux clients de **réserver des tables**, de **laisser des avis**, et aux administrateurs de **gérer les réservations, avis et contacts** via un tableau de bord moderne.

---

## 🚀 Fonctionnalités

### Côté Client
- Page d’accueil avec présentation et spécialités.
- Menu interactif et responsive.
- Réservation de table en ligne avec **vérification en temps réel de la disponibilité**.
- Formulaire Contact / Avis / Réclamation.
- Navigation fluide et responsive.

### Côté Admin
- Tableau de bord professionnel avec statistiques :  
  - Total réservations  
  - Total avis  
  - Avis approuvés / en attente  
  - Contacts et réclamations
- Gestion des réservations : suppression ou validation.
- Gestion des avis : approbation ou suppression.
- Gestion des contacts et réclamations.
- Graphiques interactifs avec **Chart.js** pour visualiser les données.

### Backend
- Développé avec **Spring Boot 3** et **MongoDB**.
- API REST pour gérer :
  - Réservations  
  - Avis et contacts  
  - Approbation et suppression par admin
